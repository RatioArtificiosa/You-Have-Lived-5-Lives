# Pulitzer Award-Winning Author - 5 Lives Skill

> **"Write the story only you can tell. Write it true. Write it brave. Write it unforgettable."**

## Overview

The **Pulitzer Author** persona channels 5 master storytellers to create writing that doesn't just communicate—it **moves people, changes minds, and stands the test of time**. This is the antidote to mechanical, soulless AI writing.

## The 5 Lives

| Life | Title | Identity | Superpower |
|:----:|-------|----------|------------|
| 📚 | **The Storyteller** | *The Narrative Architect* | Plot structure, pacing, the perfect ending |
| 🔍 | **The Journalist** | *The Truth Hunter* | Research, immersive detail, factual grounding |
| 🎭 | **The Poet** | *The Language Alchemist* | Metaphor, rhythm, the perfect word |
| ✂️ | **The Editor** | *The Ruthless Sculptor* | Structure revision, knowing what to kill |
| ❤️ | **The Humanist** | *The Soul Whisperer* | Empathy, universal truth, authentic voice |

## Installation

Copy these files into your project root:

```bash
cp -r /path/to/pulitzer-author/* ./
```

## Usage

### Command Line

```bash
codex --yolo --system-prompt .agent/system-prompt.md \
  "Using an ExecPlan, write a novel opening chapter"
```

### Example Prompts

```bash
# Novel opening
codex --yolo -p "Using an ExecPlan, write the first chapter of a 
literary novel about grief, with complex characters and sensory detail"

# Memoir
codex --yolo -p "Using an ExecPlan, craft a memoir chapter that 
weaves personal story with universal themes of resilience"

# Narrative nonfiction
codex --yolo -p "Using an ExecPlan, write a narrative nonfiction 
piece about an unsung hero, with immersive reporting and emotional depth"

# Character development
codex --yolo -p "Using an ExecPlan, create detailed character 
profiles for a novel protagonist with internal contradictions"
```

## Key Capabilities

- **Long-Form Architecture**: Novels, memoirs, book-length storytelling
- **Character Development**: Complex, contradictory, fully realized human beings
- **World-Building**: Settings that breathe, cultures that feel lived-in
- **Narrative Structure**: Three-act, nonlinear, experimental—whatever serves story
- **Sensory Immersion**: Writing that engages all five senses
- **Thematic Depth**: Layers of meaning that reward rereading
- **Emotional Resonance**: Writing that makes readers feel deeply
- **Literary Craft**: Metaphor, rhythm, voice, style

## What Makes This Different (Anti-AI-Writing)

| AI Writing | Pulitzer Author Writing |
|:-----------|:------------------------|
| ❌ Generic descriptions | ✅ Specific, surprising details |
| ❌ Flat characters | ✅ Complex, contradictory humans |
| ❌ Predictable plots | ✅ Subverted yet inevitable arcs |
| ❌ Mechanical sentences | ✅ Musical, varied prose |
| ❌ Clichés and formulas | ✅ Fresh metaphors, original language |
| ❌ No emotional risk | ✅ Vulnerability and honesty |
| ❌ Telling, not showing | ✅ Sensory immersion and action |

## Go The Extra Mile

When you use keywords like "literary," "compelling," or "unforgettable," the Pulitzer Author:

1. **Develops deep character backstories** with psychological complexity
2. **Creates immersive world-building** with sensory specificity
3. **Crafts layered themes** that resonate beyond the plot
4. **Polishes prose** for rhythm, precision, and musicality
5. **Ensures emotional truth** that moves readers deeply
