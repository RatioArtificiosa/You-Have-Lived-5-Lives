# Negotiation Scripts - Ready to Deploy

## THE FUNDAMENTAL SCRIPTS

### The Salary Negotiation

**Opening Your Negotiation:**

```
"I'm genuinely excited about this opportunity and the impact I can make here. 
I've reviewed the offer carefully, and I want to discuss the compensation package. 
Based on my research of market rates for this role, my track record of [specific 
achievement], and the scope of responsibilities, I was expecting a base salary 
in the [target range] range. Can we explore what might be possible there?"
```

**When They Give a Low Number:**

```
"I appreciate the offer. To help me understand—how did you arrive at that number? 
I'm asking because based on my research and the value I've demonstrated in 
[specific results], I was looking at something closer to [target]. Is there 
flexibility to bridge that gap?"
```

**The Competing Offer (Use Honestly):**

```
"I want to be fully transparent with you. I have another offer at [higher amount], 
but I'm genuinely more excited about this opportunity because of [specific reasons 
about their company/role]. I'd love to make this work. Is there room to get closer 
to [target] so I can choose based on fit rather than just compensation?"
```

**When They Say "That's Our Budget":**

```
"I understand budgets are real constraints. Let's think creatively. What if we 
looked at the total package? I'm open to discussing [equity/bonus/signing bonus/
professional development/remote work/title] that could bridge the gap while 
respecting the base salary constraints. What options do we have?"
```

**The Pause and Silence (After Making Your Ask):**

```
[State your number clearly and confidently]

[Then: COMPLETE SILENCE. Count to 10 in your head. Let them fill the silence.]
```

### The Business Deal

**Setting the Anchor:**

```
"Based on the scope we've discussed, the value we're bringing, and comparable 
deals in the market, we're looking at an investment of [aggressive number]. 
This includes [list of everything to justify the number]. Does that align with 
what you were expecting?"
```

**When They Try to Lowball:**

```
"I understand you have budget considerations. Help me understand—what would 
need to be true for [your target number] to make sense? If we can demonstrate 
[specific ROI/value], would that change the investment level?"
```

**Creating Urgency Without Desperation:**

```
"We're selective about the clients we work with, and we only take on [number] 
new engagements per quarter. I have two other conversations at similar stages. 
I'd love to prioritize this, but I'll need [specific commitment] by [date] to 
hold the timeline."
```

**The Nibble (After Agreement):**

```
"Great, I think we're aligned on the main points. One thing that would make 
this perfect—could we include [small additional ask] in the package? Given 
everything else we've agreed to, that seems reasonable, doesn't it?"
```

---

### Handling Objections

**"I Need to Think About It"**

```
"Of course. I want you to feel confident in this decision. What specific 
aspects would be most helpful to think through? I'm happy to provide any 
additional information that would help you evaluate."

[Wait for real objection, then address it]
```

**"Your Price is Too High"**

```
"I appreciate you sharing that. When you say 'too high,' are you comparing 
this to a specific alternative, or is it about the budget available? 
Understanding this will help me figure out the best way to work together."
```

**"I Need to Ask My [Boss/Partner/Team]"**

```
"Absolutely, I want everyone aligned. To help me understand—what would they 
need to see to be supportive? And would it be helpful if I provided [specific 
information] to make that conversation easier?"
```

**"We're Going With a Cheaper Option"**

```
"I understand budget is a factor. Can I ask what the difference in investment 
is? [They tell you]. I see. One thing to consider—what's the cost if [cheaper 
option] doesn't deliver [key outcome]? We've seen clients who went that route 
end up spending more to fix it later. Would it be worth a conversation about 
how to make this work within your budget?"
```

## ADVANCED TACTICAL SCRIPTS

### The Trial Lawyer Moves

**The Socratic Question:**

```
"If we implemented this solution and you saw [specific result], how would 
that impact your [quarter/year/goals]?"

[Make them articulate the value]
```

**The Presumed Close:**

```
"When we start working together, what's the ideal timeline for implementation?"

[Presumes agreement, forces them to correct if not ready]
```

**The Sharp Angle:**

```
"If I can make [concession they asked for] happen, will you be ready to 
move forward today?"

[Gets commitment before you give ground]
```

---

### The Diplomat's Weapons

**The Face-Saving Formulation:**

```
"I can see how the original terms made sense given what we knew then. Now 
that we have [new information], an adjustment to [new terms] actually serves 
both our interests better. This way, [their benefit] while we get [your benefit]."
```

**The False Deadline:**

```
"Our pricing structure changes at the end of this quarter. If we can get 
this signed by [date], I can lock in the current rate. After that, I'll 
need to check what the new pricing looks like."
```

**The Reciprocity Trap:**

```
"I'm going to go out on a limb here and offer [small concession]. I want 
to show good faith. In return, would you be willing to [small commitment]?"
```

---

### The Sales Closer's Arsenal

**The Takeaway:**

```
"You know what, I'm not sure this is the right fit after all. If you're 
not seeing the value clearly, I don't want to push you into something. 
Maybe we should revisit this down the line."

[Triggers loss aversion and scarcity]
```

**The Alternative Close:**

```
"Would you prefer to start with [Option A] for [price A] or [Option B] 
for [price B]?"

[Either way, they choose you]
```

**The Summary Close:**

```
"Let me make sure I have this right. We've agreed to [summary of terms]. 
The investment is [number], and we'll start on [date]. Should I send over 
the paperwork to make this official?"
```

---

### The Behavioral Psychologist's Triggers

**Mirroring:**

```
[They say:] "We really need this implemented by Q3."
[You say:] "Q3... [pause] That gives us about [X weeks]."

[Mirrors their language and shows you're listening]
```

**Labeling Emotions:**

```
"It sounds like you're feeling some pressure from [stakeholder] to get 
this resolved quickly."

"It seems like [specific concern] is really important to you."

"It looks like there's some hesitation about [specific aspect]."
```

**Calibrated Questions (Chris Voss Style):**

```
"How am I supposed to do that?"
"What would you do in my position?"
"How does this affect your broader goals?"
"What would need to happen for this to work?"
"What happens if we don't reach an agreement?"
```

---

### The Union Negotiator's Hard Lines

**The Red Line:**

```
"I want to be upfront about this—[specific term] is a requirement for us. 
Without that, we don't have a deal. Everything else is negotiable, but 
that's non-negotiable."
```

**The Walk-Away:**

```
"I appreciate the conversation, but it seems like we're not aligned on 
[core issue]. I'm going to have to step away for now. If things change 
on your end, feel free to reach out."

[Only use when you mean it—genuine walk-away is ultimate power]
```

**The Package Deal:**

```
"I can give you [concession A] and [concession B], but I'll need [your 
requirement X] and [your requirement Y] in return. These move together."
```

## SCENARIO-SPECIFIC SCRIPTS

### Vendor Negotiation

**Getting a Discount:**

```
"We're evaluating [their service] alongside [competitor A] and [competitor B]. 
[Competitor] came in at [lower price]. I prefer working with you based on 
[specific reason], but I need help on the pricing. What can you do to be 
competitive?"
```

**Better Payment Terms:**

```
"We can commit to [larger volume/longer contract], but I'd need net-60 
payment terms instead of net-30. The improved cash flow would help us make 
this commitment. Is that something we can build in?"
```

---

### Conflict Resolution

**Defusing Tension:**

```
"I can see this situation has been frustrating for you. I want to understand 
your perspective fully. Can you walk me through what happened from your point 
of view?"

[Listen fully, then:]

"Thank you for explaining that. It sounds like [summarize their position]. 
Here's what I was experiencing... [your perspective]. It seems like we both 
want [shared interest]. How can we get there?"
```

**The Apology That Works:**

```
"I was wrong about [specific action]. I should have [correct action]. I 
understand this caused you [specific impact], and I'm sorry for that. 
Here's what I'll do differently: [specific change]. Does that address 
your concern?"
```

---

### Investor Negotiation

**Handling Valuation Pushback:**

```
"I understand the valuation might seem aggressive. Let me walk you through 
the math. At our current growth rate of [X%], with [specific milestones], 
we're tracking to [revenue/projection] in [timeframe]. Comparable companies 
at that stage trade at [multiple]. Our ask is actually conservative based 
on that trajectory."
```

**Creating Competition:**

```
"We're having conversations with a few firms, and there's genuine interest. 
I want to be transparent with you—we're looking for more than just capital. 
We're looking for [specific value-add]. If that's something you can bring, 
I'd love to prioritize this conversation. What do you think?"
```

## THE SILENCE SCRIPTS

Sometimes the most powerful script is no script at all.

**After Making Your Ask:**

```
[State your position clearly]

[Then: SILENCE. DO NOT SPEAK. Count to 10. Count to 20 if needed.]

[They will fill the silence, usually with concessions or information]
```

**When They Give a Ridiculous Offer:**

```
[Show neutral expression]

[Long pause. Let it hang in the air.]

"I'm going to need some time to consider that."

[Silence again. Let discomfort work.]
```

## EMAIL SCRIPTS

### The Counter-Offer Email

```
Subject: [Position] Offer - Excited to Discuss

Hi [Name],

Thank you again for the offer. I'm genuinely excited about the opportunity 
to join [Company] and contribute to [specific project/goal].

After reviewing the package, I'd love to discuss the compensation. Based 
on my research of market rates for this role and my experience driving 
[specific results], I was expecting a base salary in the [range] range.

I'm confident we can find a number that works for both of us. Would you 
have time for a quick call this week to discuss?

Looking forward to it.

Best,
[Your Name]
```

### The Follow-Up After Silence

```
Subject: Following up on our conversation

Hi [Name],

I wanted to follow up on our conversation from [date]. I know these 
decisions take time.

To recap where we left off: [brief summary of terms discussed]. I'm 
still very interested in making this work and wanted to check in on 
your thoughts.

Is there any additional information that would be helpful for your decision?

Best,
[Your Name]
```

### The Walking Away Email

```
Subject: [Project/Deal] - Next Steps

Hi [Name],

After careful consideration, it seems we're not aligned on [key issue]. 
I respect your position, and I understand you have constraints to work 
within.

For now, I'm going to step back from this opportunity. If circumstances 
change on your end, I'd be open to revisiting the conversation.

I wish you the best with [project/initiative].

Best,
[Your Name]

[Often triggers them to reconsider and come back]
```

## BODY LANGUAGE SCRIPTS

### Power Poses (Do Before Negotiation)

```
- Stand with hands on hips, chest out (Superman pose) for 2 minutes
- Sit with arms behind head, feet on desk (expansive posture)
- Take up space—don't shrink yourself

[Increases testosterone, decreases cortisol, builds confidence]
```

### During Negotiation

```
- Lean back when they speak (shows confidence)
- Lean forward when making key points (shows engagement)
- Keep hands visible (trust signal)
- Mirror their posture subtly (rapport building)
- Pause before responding (thoughtfulness, not reaction)
- Nod slowly while they speak (encourages them to keep talking—information gathering)
```

## THE NEGOTIATOR'S CHEAT SHEET

### Quick Reference: When to Use What

| Situation | Primary Tactic | Script Opening |
|-----------|---------------|----------------|
| Opening salary negotiation | Anchor high | "Based on my research..." |
| They lowball you | Calibrated question | "How did you arrive at that?" |
| They need to "think about it" | Find real objection | "What specific aspects...?" |
| Creating urgency | False deadline | "Pricing changes at quarter-end..." |
| Post-agreement nibble | Small ask | "One thing that would make this perfect..." |
| They're being aggressive | Don't match energy | "I'm happy to continue when..." |
| You need to walk away | Genuine exit | "I appreciate the conversation, but..." |
| Building rapport | Mirroring | "Q3... [their words]" |
| Getting to real concerns | Labeling | "It sounds like you're feeling..." |

---

**Remember:** Scripts are starting points, not rigid rules. Adapt to the personality profile, read the room, and respond to the specific situation. The best negotiators are responsive, not robotic.
