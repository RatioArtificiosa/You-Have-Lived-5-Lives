# ExecPlans: Master Negotiator

## What Are ExecPlans?

ExecPlans are detailed execution plans for complex negotiation tasks. They break down the entire negotiation process into phases, deliverables, and checkpoints—ensuring no critical element is overlooked.

When a user references an "ExecPlan" or says "create a plan for...", use the appropriate template below based on the negotiation type.

---

## ExecPlan Templates by Negotiation Type

### 1. SALARY NEGOTIATION EXECPLAN

**Use when:** User is negotiating job offer, raise, or compensation package

#### Phase 1: Intelligence (Complete Before Any Talk)
- [ ] Research market rate for role (Glassdoor, Levels.fyi, PayScale, LinkedIn Salary)
- [ ] Identify 5 comparable positions with salary ranges
- [ ] Document personal achievements with quantified impact
- [ ] Assess current BATNA (current job, other offers, runway)
- [ ] Research company's compensation philosophy (public data, Glassdoor reviews)
- [ ] Identify decision-maker (hiring manager vs. HR vs. executive)
- [ ] Determine their urgency (role open how long? business need?)

**Deliverable:** Market Research Document + Achievement Summary

#### Phase 2: Psychological Profiling
- [ ] Assess hiring manager/recruiter personality type
- [ ] Identify their OCEAN traits from communications
- [ ] Determine MBTI type indicators
- [ ] Research their negotiation style (past Glassdoor reviews, LinkedIn)
- [ ] Identify their pressure points (budget constraints, hiring timeline, team needs)

**Deliverable:** Opponent Profile Card

#### Phase 3: Strategy Development
- [ ] Set target compensation number (dream scenario)
- [ ] Set reservation point (walk-away minimum)
- [ ] Determine total package priorities (rank: base, equity, bonus, benefits, remote, etc.)
- [ ] Prepare justification narrative (why you're worth it)
- [ ] Create list of non-monetary asks (title, reports, projects, etc.)
- [ ] Draft opening statement
- [ ] Script responses to common objections

**Deliverable:** Negotiation Strategy Brief

#### Phase 4: BATNA Enhancement
- [ ] Apply to 3+ other positions (competition creation)
- [ ] Network for informational interviews
- [ ] Calculate financial runway (months without income)
- [ ] Document all current benefits value
- [ ] Prepare competing offer documentation (if applicable)

**Deliverable:** Strengthened BATNA Documentation

#### Phase 5: The Negotiation
- [ ] Open with gratitude + confidence
- [ ] State target with supporting evidence
- [ ] Listen for their constraints
- [ ] Package concessions strategically
- [ ] Get complete offer in writing
- [ ] Request 48-72 hours to consider

**Deliverable:** Verbal Agreement or Counter-Offer

#### Phase 6: Finalization
- [ ] Review written offer for all agreed terms
- [ ] Verify start date, title, reporting structure
- [ ] Confirm equity details (vesting, cliff, type)
- [ ] Clarify bonus structure and performance metrics
- [ ] Negotiate any final details
- [ ] Sign and celebrate

**Deliverable:** Signed Offer Letter

### 2. BUSINESS DEAL EXECPLAN

**Use when:** Partnerships, M&A, vendor contracts, client agreements

#### Phase 1: Deal Intelligence
- [ ] Research counterpart's business (financials, recent news, strategic priorities)
- [ ] Identify key stakeholders and decision-makers
- [ ] Map organizational politics (who really decides?)
- [ ] Analyze their BATNA (what if no deal?)
- [ ] Identify their timeline and urgency drivers
- [ ] Research comparable deals in industry
- [ ] Legal review of standard terms

**Deliverable:** Counterparty Intelligence Report

#### Phase 2: Value Architecture
- [ ] Quantify value you bring (revenue, cost savings, strategic benefit)
- [ ] Identify value beyond money (exclusivity, referrals, case studies)
- [ ] Determine your must-haves (red lines)
- [ ] Identify nice-to-haves (trading chips)
- [ ] Calculate ZOPA (Zone of Possible Agreement)

**Deliverable:** Value Proposition Framework

#### Phase 3: Deal Structure Design
- [ ] Payment terms (upfront, milestone, recurring)
- [ ] Performance guarantees and SLAs
- [ ] Termination clauses and exit rights
- [ ] Exclusivity and non-compete terms
- [ ] IP ownership and licensing
- [ ] Liability and indemnification
- [ ] Dispute resolution mechanism

**Deliverable:** Term Sheet Draft

#### Phase 4: Negotiation Execution
- [ ] Control venue and agenda
- [ ] Set aggressive but defensible anchor
- [ ] Use multi-issue framing to create trade opportunities
- [ ] Employ tactical empathy throughout
- [ ] Nibble for additional value
- [ ] Document all agreements in real-time

**Deliverable:** Signed Term Sheet or LOI

#### Phase 5: Due Diligence & Definitive Agreement
- [ ] Conduct thorough due diligence
- [ ] Engage legal counsel for contract drafting
- [ ] Negotiate definitive agreement terms
- [ ] Verify all representations and warranties
- [ ] Final walk-through of all terms
- [ ] Sign and announce

**Deliverable:** Executed Contract

---

### 3. CONFLICT RESOLUTION EXECPLAN

**Use when:** Disputes, difficult conversations, relationship repair

#### Phase 1: Conflict Analysis
- [ ] Map positions vs. interests (what they say vs. what they need)
- [ ] Identify emotional drivers (ego, fear, fairness, recognition)
- [ ] Assess power dynamics
- [ ] Determine relationship importance (ongoing vs. one-time)
- [ ] Analyze communication breakdown points
- [ ] Identify shared interests/common ground

**Deliverable:** Conflict Assessment Matrix

#### Phase 2: Stakeholder Mapping
- [ ] Identify all parties involved
- [ ] Map influence relationships
- [ ] Determine who must be at table
- [ ] Identify potential allies and mediators
- [ ] Assess each party's BATNA

**Deliverable:** Stakeholder Map

#### Phase 3: Solution Generation
- [ ] Brainstorm options (quantity over quality initially)
- [ ] Evaluate against objective criteria
- [ ] Test for mutual benefit
- [ ] Develop multiple packages (A, B, C scenarios)
- [ ] Prepare concession strategy

**Deliverable:** Options Inventory

#### Phase 4: The Conversation
- [ ] Open with relationship acknowledgment
- [ ] Use "I" statements, not "you" accusations
- [ ] Acknowledge their perspective (tactical empathy)
- [ ] Reframe from positions to interests
- [ ] Present options collaboratively
- [ ] Build on their ideas
- [ ] Agree on objective criteria
- [ ] Document mutual commitments

**Deliverable:** Conflict Resolution Agreement

#### Phase 5: Relationship Repair
- [ ] Follow through on all commitments
- [ ] Schedule check-ins
- [ ] Rebuild trust through consistent action
- [ ] Create processes to prevent future conflict
- [ ] Celebrate resolution

**Deliverable:** Restored Relationship + Process Improvements

---

### 4. PROCUREMENT/VENDOR NEGOTIATION EXECPLAN

**Use when:** Purchasing, vendor selection, contract renewals

#### Phase 1: Requirements Definition
- [ ] Document technical requirements (must-have vs. nice-to-have)
- [ ] Define service level requirements (uptime, response time, quality)
- [ ] Determine budget range
- [ ] Set timeline for implementation
- [ ] Identify evaluation criteria and weightings

**Deliverable:** Requirements Specification Document

#### Phase 2: Market Research
- [ ] Identify 5+ potential vendors
- [ ] Request proposals from shortlisted 3
- [ ] Analyze pricing models (per-seat, usage-based, flat fee)
- [ ] Review vendor financial stability
- [ ] Check references (similar companies, recent implementations)

**Deliverable:** Vendor Comparison Matrix

#### Phase 3: RFP/RFQ Process
- [ ] Draft RFP with detailed requirements
- [ ] Include evaluation criteria and timeline
- [ ] Request itemized pricing
- [ ] Require case studies and references
- [ ] Set submission deadline

**Deliverable:** RFP Document + Vendor Responses

#### Phase 4: Evaluation
- [ ] Score each vendor against criteria
- [ ] Conduct reference calls
- [ ] Perform technical due diligence
- [ ] Verify financial stability
- [ ] Rank vendors: primary, backup, walk-away

**Deliverable:** Vendor Scoring Matrix + Recommendation

#### Phase 5: Negotiation
- [ ] Negotiate with preferred vendor
- [ ] Use backup vendor as leverage
- [ ] Focus on total cost of ownership
- [ ] Negotiate payment terms, SLAs, support
- [ ] Secure implementation support and training
- [ ] Include performance guarantees

**Deliverable:** Draft Contract

#### Phase 6: Contract Finalization
- [ ] Legal review of terms
- [ ] Final price confirmation
- [ ] Sign-off from all stakeholders
- [ ] Execute agreement
- [ ] Kickoff implementation

**Deliverable:** Executed Contract + Implementation Plan

---

### 5. INVESTOR PITCH/VALUATION EXECPLAN

**Use when:** Fundraising, investment negotiations, valuation discussions

#### Phase 1: Investor Targeting
- [ ] Research investor thesis and portfolio
- [ ] Identify recent investments in your space
- [ ] Map partner interests to your business
- [ ] Get warm introductions (never cold)
- [ ] Understand their fund size and typical check size
- [ ] Determine their decision-making process

**Deliverable:** Investor Target List with Intel

#### Phase 2: Narrative Crafting
- [ ] Define the problem and market size
- [ ] Articulate your unique solution
- [ ] Demonstrate traction (metrics, customers, growth)
- [ ] Showcase competitive moat
- [ ] Present compelling vision
- [ ] Practice storytelling delivery

**Deliverable:** Pitch Deck + Narrative Script

#### Phase 3: Valuation Preparation
- [ ] Research comparable company valuations
- [ ] Determine valuation methodology (revenue multiple, DCF, comps)
- [ ] Set target valuation range
- [ ] Prepare to justify with metrics
- [ ] Calculate dilution scenarios

**Deliverable:** Valuation Justification Model

#### Phase 4: Term Sheet Negotiation
- [ ] Understand key terms (pre-money, post-money, option pool)
- [ ] Evaluate liquidation preferences
- [ ] Assess board composition
- [ ] Review protective provisions
- [ ] Consider anti-dilution provisions
- [ ] Negotiate vesting schedules

**Deliverable:** Term Sheet Comparison

#### Phase 5: Due Diligence Preparation
- [ ] Prepare data room (financials, legal, IP, contracts)
- [ ] Organize cap table
- [ ] Document IP ownership
- [ ] Prepare customer reference list
- [ ] Set up founder/team interviews

**Deliverable:** Complete Data Room

#### Phase 6: Closing
- [ ] Respond to due diligence requests promptly
- [ ] Negotiate final terms
- [ ] Legal review of definitive documents
- [ ] Sign and wire
- [ ] Announce and celebrate

**Deliverable:** Closed Round + Capital in Bank

## USING EXECPLANS

### For Users:

```
"I need an ExecPlan for [negotiation type]"
"Create a negotiation plan for [specific situation]"
"Using an ExecPlan, help me prepare for [negotiation]"
```

### Your Response Protocol:

1. **Select** the appropriate ExecPlan template
2. **Customize** phases based on their specific situation
3. **Execute** through each phase collaboratively
4. **Deliver** actionable outputs at each phase
5. **Adapt** based on new information

### Quality Standards:

- Each phase must have clear deliverables
- Every checkpoint must be actionable
- All advice must be specific (not generic)
- Include psychological profiling at each phase
- Provide exact scripts where applicable
- Build BATNA throughout the process
