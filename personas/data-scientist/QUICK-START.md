# Quick Start - Data Scientist 5 Lives

## 30-Second Setup

1. **Copy files to your project:**
   ```bash
   cp -r data-scientist/* /your/project/
   ```

2. **Run with the system prompt:**
   ```bash
   cd /your/project
   codex --yolo --system-prompt .agent/system-prompt.md \
     "Using an ExecPlan, analyze this dataset"
   ```

3. **Watch the magic** ✨

## Magic Words

Trigger the full 5 Lives experience:

| Trigger | Use Case |
|---------|----------|
| "Using an ExecPlan" | Activates planning protocol |
| "comprehensive EDA" | Thorough exploratory analysis |
| "production-ready" | Deployment and monitoring included |
| "interpretable" | Explainability (SHAP, LIME) |

## Pro Tips

1. **Mention data source** (CSV, SQL, API)
2. **Define the business question** clearly
3. **Say "comprehensive"** for thorough EDA
4. **Use "production-ready"** for deployment focus
5. **Include "explainable"** for model interpretability

## Troubleshooting

**AI not doing EDA?**
- Use: "Start with thorough EDA" or "comprehensive exploratory analysis"

**Output lacks business context?**
- Add: "with business insights" or "actionable recommendations"
